# PhotoCleaner 디자인 공통 언어

> 앱 전반에서 일관된 용어를 사용하여 사용자 혼란을 방지하고 명료한 커뮤니케이션을 제공한다.

---

## 1. 핵심 개체 용어

### 1.1 사진 관련

| 공식 용어 | 사용 금지 | 설명 |
|----------|----------|------|
| **사진** | 이미지, 이미지 파일, 미디어, 에셋 | 사용자가 촬영하거나 저장한 사진 |
| **동영상** | 비디오, 영상, 미디어 | 사용자가 촬영하거나 저장한 동영상 |
| **항목** | 아이템, 에셋, 파일 | 사진과 동영상을 통칭할 때 |
| **원본** | 오리지널, 풀 사이즈 | iCloud에 저장된 전체 해상도 파일 |
| **썸네일** | 미리보기, 프리뷰 | 축소된 미리보기 이미지 |

### 1.2 공간/위치 관련

| 공식 용어 | 사용 금지 | 설명 |
|----------|----------|------|
| **사진 보관함** | 라이브러리, 갤러리, 앨범 | 전체 사진 저장소 |
| **앨범** | 폴더, 컬렉션 | 사용자가 만든 사진 그룹 |
| **iCloud** | 클라우드, 서버 | Apple의 클라우드 저장소 |
| **기기** | 디바이스, 로컬, 폰 | 현재 사용 중인 iPhone/iPad |
| **최근 삭제된 항목** | 휴지통, 쓰레기통, 삭제됨 | 삭제 후 복구 가능 영역 (시스템 명칭 그대로) |

## 2. 문제 유형별 명칭

### 2.1 공식 문제 유형

| 코드명 | UI 표시명 | 짧은 설명 | 상세 설명 |
|--------|----------|----------|----------|
| `needsDownload` | **다운로드 필요** | iCloud에서 가져오기 | 원본이 iCloud에만 있어 기기로 다운로드해야 합니다 |
| `downloadFailed` | **다운로드 실패** | 가져올 수 없음 | iCloud에서 사진을 가져오지 못했습니다 |
| `corrupted` | **손상됨** | 파일이 손상됨 | 사진 파일이 손상되어 볼 수 없습니다 |
| `duplicate` | **중복** | 같은 사진 존재 | 동일하거나 매우 유사한 사진이 있습니다 |
| `lowQuality` | **저화질** | 원본보다 낮은 품질 | 원본보다 해상도가 낮은 사진입니다 |
| `screenshot` | **스크린샷** | 화면 캡처 | 화면을 캡처한 이미지입니다 |
| `largeFile` | **용량 큼** | 많은 저장 공간 사용 | 파일 크기가 큰 사진입니다 |

### 2.2 문제 심각도

| 레벨 | 용어 | 색상 | 설명 |
|-----|------|------|------|
| 1 | **심각** | 빨강 | 즉시 조치 필요 (손상, 다운로드 실패) |
| 2 | **주의** | 주황 | 조치 권장 (다운로드 필요) |
| 3 | **참고** | 파랑 | 선택적 조치 (중복, 저화질) |
| 4 | **정보** | 회색 | 단순 정보 (스크린샷, 용량) |

## 3. 액션 동사

### 3.1 기본 액션

| 공식 용어 | 사용 금지 | 의미 |
|----------|----------|------|
| **삭제** | 제거, 지우기, 휴지통으로 | 사진을 최근 삭제된 항목으로 이동 |
| **영구 삭제** | 완전 삭제, 영원히 삭제 | 복구 불가능하게 삭제 |
| **복구** | 복원, 되살리기 | 삭제된 사진을 원래 위치로 |
| **다운로드** | 가져오기, 내려받기 | iCloud에서 기기로 가져오기 |
| **검사** | 스캔, 분석, 체크 | 사진 보관함 문제 확인 |
| **선택** | 체크, 마크 | 항목을 선택 상태로 |
| **선택 해제** | 언체크, 마크 해제 | 선택 상태 취소 |

### 3.2 일괄 액션

| 공식 용어 | 사용 금지 | 의미 |
|----------|----------|------|
| **모두 선택** | 전체 선택, 일괄 선택 | 표시된 모든 항목 선택 |
| **선택 초기화** | 전체 해제, 모두 해제 | 모든 선택 취소 |
| **일괄 삭제** | 전체 삭제, 한번에 삭제 | 선택한 여러 항목 삭제 |
| **추천 선택** | AI 선택, 자동 선택 | 삭제 추천 항목 자동 선택 |

### 3.3 설정 액션

| 공식 용어 | 의미 |
|----------|------|
| **켜기** / **끄기** | 토글 설정 변경 |
| **변경** | 값 수정 |
| **초기화** | 기본값으로 되돌림 |

## 5. 메시지 템플릿

### 5.1 확인 대화상자

```
[삭제 확인]
제목: "{N}장의 사진을 삭제할까요?"
내용: "이 사진들은 '최근 삭제된 항목'으로 이동되며, 30일 후 영구 삭제됩니다."
버튼: [취소] [삭제]

[영구 삭제 확인]
제목: "{N}장의 사진을 영구 삭제할까요?"
내용: "이 동작은 되돌릴 수 없습니다."
버튼: [취소] [영구 삭제]
```

### 5.2 완료 메시지

```
[삭제 완료]
"{N}장의 사진이 삭제되었습니다"

[다운로드 완료]
"{N}장의 사진을 다운로드했습니다"

[검사 완료]
"검사 완료: 문제 {N}개 발견"
또는
"검사 완료: 문제 없음"
```

### 5.3 오류 메시지

```
[일반 오류]
제목: "문제가 발생했어요"
내용: "{구체적 설명}"
버튼: [확인] 또는 [다시 시도]

[네트워크 오류]
제목: "인터넷 연결을 확인해 주세요"
내용: "iCloud에 연결하려면 인터넷이 필요합니다."
버튼: [확인]

[권한 오류]
제목: "사진 접근 권한이 필요해요"
내용: "설정에서 PhotoCleaner의 사진 접근을 허용해 주세요."
버튼: [취소] [설정으로 이동]
```

### 5.4 빈 상태 메시지

```
[문제 없음]
아이콘: 체크마크
제목: "모든 사진이 깨끗해요"
내용: "문제가 발견되지 않았습니다."

[사진 없음]
아이콘: 사진 아이콘
제목: "사진이 없어요"
내용: "사진을 추가하면 여기에 표시됩니다."

[검사 필요]
아이콘: 검색 아이콘
제목: "검사를 시작해 주세요"
내용: "사진 보관함을 검사하여 문제를 찾습니다."
버튼: [검사 시작]
```

## 6. 수량 표현

### 6.1 숫자 포맷

| 규칙 | 예시 |
|-----|------|
| 1,000 미만 | "127장" |
| 1,000 이상 | "1,234장" |
| 10,000 이상 | "12,345장" |
| 개수 0 | "0장" (숨기지 않음) |

### 6.2 단위 표현

| 항목 | 단위 | 예시 |
|-----|------|------|
| 사진/동영상 개수 | 장 | "15장" |
| 파일 크기 | KB, MB, GB | "1.2 GB" |
| 시간 | 초, 분, 시간 | "약 2분" |
| 날짜 | 상대/절대 | "2분 전", "2024년 1월 1일" |

| 경과 시간 | 표시 |
|----------|------|
| 1분 미만 | "방금" |
| 1-59분 | "{N}분 전" |
| 1-23시간 | "{N}시간 전" |
| 1-6일 | "{N}일 전" |
| 7일 이상 | "YYYY년 M월 D일" |

## 7. 접근성 레이블

### 7.1 버튼

```swift
// 삭제 버튼
.accessibilityLabel("선택한 사진 삭제")
.accessibilityHint("15장의 사진을 최근 삭제된 항목으로 이동합니다")

// 선택 버튼
.accessibilityLabel("사진 선택")
.accessibilityValue(isSelected ? "선택됨" : "선택 안 됨")
```

### 7.2 상태 배지

```swift
// 문제 개수 배지
.accessibilityLabel("다운로드 필요한 사진 23장")

// 진행률
.accessibilityLabel("검사 진행률 35퍼센트")
```

### 7.3 리스트 항목

```swift
// 사진 셀
.accessibilityLabel("사진, 2024년 1월 1일 촬영, 다운로드 필요")
.accessibilityHint("두 번 탭하여 상세 보기")
```

## 9. 피해야 할 표현

### 9.1 기술 용어

| 사용 금지 | 대체 표현 |
|----------|----------|
| PHAsset | 사진/동영상 |
| iCloud 동기화 대기 중 | 다운로드 필요 |
| 메타데이터 | 사진 정보 |
| 캐시 | 임시 저장 데이터 |
| 에러 코드 -1 | 문제가 발생했어요 |

### 9.2 부정확한 표현

| 사용 금지 | 이유 |
|----------|------|
| "삭제합니다" (단정) | 사용자 확인 전 사용 금지 |
| "영구 삭제됩니다" | 최근 삭제로 먼저 이동함을 명시 |
| "완전히 제거" | "영구 삭제"로 통일 |

### 9.3 모호한 표현

| 사용 금지 | 대체 표현 |
|----------|----------|
| "일부" | 구체적 숫자 |
| "많은" | 구체적 숫자 |
| "잠시 후" | "약 2분" 등 구체적 시간 |
| "오류가 발생" | 구체적 원인과 해결책 |
